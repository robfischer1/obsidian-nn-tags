# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.2.x   | :white_check_mark: |
| < 1.2   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by opening a
[GitHub issue](https://github.com/robfischer1/obsidian-nn-tags/issues) or
contacting the maintainer via GitHub.

You can expect an initial response within 7 days. If the vulnerability is
accepted, a fix will be prioritized and released as a patch version. If
declined, an explanation will be provided.
